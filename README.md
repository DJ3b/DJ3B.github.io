# Portfolio - Said AFRITE

Personal portfolio (FR/EN) showcasing my mobile-robotics internship projects:
robust EKF target tracking (LiDAR + AprilTag ground truth), global localization
(K-Means + Procrustes, Gauss-Newton), SLAM, embedded programming (mBot2/CyberPi)
and ROS 2 / Docker infrastructure on a ROSbot 3 Pro.

Design based on the [vCard personal portfolio template](https://github.com/codewithsadee/vcard-personal-portfolio)
by codewithsadee (MIT license), adapted: bilingual FR/EN toggle, robotics content,
project articles, no contact form.
